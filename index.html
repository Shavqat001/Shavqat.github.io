Проекты

Разрабатываю решения для бизнеса и пользователей на протяжении последних 4+ лет. Превращаю идеи в работающие продукты, создавая адаптивные и интуитивные веб-приложения, которые решают реальные задачи.

  • AI/ML

    Universal Messenger

    Universal Messenger — это чат-приложение, разработанное для удобного общения между пользователями через популярными мессенджерами Телеграм и Ватсап. Он настроен через телеграм-бота и реального номера из ватсап. Пользователь написавщий сообщение боту появляется в веб-интерфейсе. Проект включает в себя веб-сокет, експресс и базу данных где сообщения сохраняются.

  • Productivity

    To-Do List

    Этот To-Do List позволяет пользователям добавлять, удалять и отмечать задачи как выполненные. Список поддерживает localStorage, благодаря чему задачи сохраняются даже после перезагрузки страницы. У каждой задачи есть иконки для отметки выполнения (галочка) и удаления (крестик). При необходимости, задачи можно редактировать. Также предусмотрена возможность добавления описания задачи, которое отображается при наведении.

  • E-Commerce

    Tekhnomart

    Техномарт — это интернет-магазин строительных материалов и инструментов для ремонта, разработанный с использованием современных веб-технологий и принципов. Сайт отличается удобной навигацией, привлекательным дизайном и высокой функциональностью, что делает его идеальным решением для клиентов, ищущих качественные товары для строительства и ремонта.
